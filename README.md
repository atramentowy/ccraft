# ccraft
A simple Minecraft clone written in C

## clone and build
clone with submodules:
```
git clone --recurse-submodules <repo-url>
```
then run:

```
sudo apt install build-essential libx11-dev libgl1-mesa-dev libglew-dev

./build.sh        # Linux
build.bat       # Windows
```

you can find executables in the ``build/`` folder
